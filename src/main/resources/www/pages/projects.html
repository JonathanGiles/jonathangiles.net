Projects

Over time I’ve worked on a number of open source projects and websites. The list below is an incomplete list based on my recollection, but you can also check out my GitHub for a number of smaller projects I created.


Site BuilderSimple static website generator / templating engine

Site Builder is a simplistic templating engine that is currently used to build JonathanGiles.net.

View on GitHub

TeenyHttpdSimple HTTP Server written in Java

TeenyHttpd is an extremely basic HTTP server. It is implemented in plain Java 8 with no external dependencies, making it lightweight and applicable in situations where a basic, small HTTP server is all that is required.

View on GitHub

ControlsFXRich UI controls library for JavaFX

I started the ControlsFX project to fill the gap between what Oracle offers in JavaFX and what the community needed. ControlsFX is an open source project for JavaFX that aims to provide really high quality UI controls and other tools to complement the core JavaFX distribution. It has been developed for JavaFX 8.0 and beyond, and has a guiding principle of only accepting new controls / features when all existing code is at an acceptably high level, including thankless jobs like having high quality javadoc documentation.

View on GitHub | Project Website



Abandoned Projects

FX ExperienceThe premiere JavaFX website

FXExperience.com is a hugely popular JavaFX website with the latest news, interviews, code tips, etc. I was the sole maintainer on this website, which received over 2000 hits a day.

Visit Website

Scenic ViewA runtime scenegraph analyser for JavaFX

Scenic View is a JavaFX application designed to make it simple to understand the current state of your application scenegraph, and to also easily manipulate properties of the scenegraph without having to keep editing your code. This lets you find bugs, and get things pixel perfect without having to do the compile-check-compile dance.

View on GitHub | Project Website

FXDocsThe JavaFX Documentation Project

The JavaFX Documentation Project is designed to collect important JavaFX information from around the web and, with the permission of the authors, to collate it into a single source to form a cohesive book.

View on GitHub | View PDF | View Website

OpenSprinklerA Java API for OpenSprinkler devices

A Java-based library for communicating with OpenSprinkler irrigation controller devices over a local area network. This is used in projects such as OpenHAB for home automation (in particular, I created an OpenHAB addon for controlling home watering automatically).
