Jonathan Giles is a Principal Software Architect at Microsoft, where he is helping to build out the next generation of Azure client libraries. Before this, Jonathan was a Java cloud developer advocate at Microsoft for one year, and a technical lead on the Java team at Sun Microsystems and Oracle for nine years, where he contributed to all Java releases between Java 7 and 10.
